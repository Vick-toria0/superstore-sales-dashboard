# superstore-sales-dashboard
Interactive sales analysis dashboard built with Excel and Tableau,
analyzing 4 years of retail transaction data (10,000+ records).

# Tools Used
- Excel — data cleaning, PivotTables, trend analysis
- Tableau Public — interactive dashboard with year filtering

# Key Insight
The South region generated higher profit margins than Central despite 
lower sales volume, suggesting a potential discounting or cost 
efficiency gap worth investigating.
